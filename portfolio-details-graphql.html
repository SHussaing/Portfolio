GraphQL Profile Page Project

This project focused on creating a profile page using the GraphQL query language. I developed a user interface that pulls personal data from a GraphQL API and displays it in an organized and visually appealing format.

The profile page includes user identification details, XP amount, grades, and audit statistics, making it easy to track progress and achievements. A unique feature of this project is the integration of SVG-based graphs, which dynamically represent statistics, such as XP earned over time and project pass/fail ratios.

The project required a JWT-based login system for secure access to data and proper user authentication. With a user-friendly login page that handles both username and email credentials, this system ensures data security and ease of use.

Hosted on GitHub Pages, the final application demonstrates a modern, responsive design that effectively showcases personal accomplishments and growth within the platform.
